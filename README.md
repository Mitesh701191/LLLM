# Universal Tracker

A flexible web application that allows you to create custom trackers for anything you want to keep track of.

## Features

- Create custom trackers with different field types (text, number, date, toggle, file, location)
- Add, edit, and delete entries
- Filter and sort entries
- View basic charts and analytics
- Export data to CSV
- User authentication with email and password

## Tech Stack

- Next.js
- Prisma
- SQLite (upgradeable to PostgreSQL)
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up the database: `npx prisma migrate dev`
4. Start the development server: `npm run dev`
5. Open [http://localhost:3000](http://localhost:3000) in your browser
